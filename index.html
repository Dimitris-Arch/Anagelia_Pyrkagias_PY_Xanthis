🚨 Tutorial: Ψηφιακή Αναγγελία Συμβάντων ΠΥ Ξάνθης

Από κουδούνι σε φωνητική αναγγελία με Raspberry Pi
5 κουμπιά συμβάντων (αναπαραγωγη TTS Google - MP3 - TTS Google), relays για φώτα/ενισχυτή. Πατώντας το άναλογο κουμπί(Παροχή βοηθείας,αστικη,δασική πυρκαγιά,τροχαίο) ενεργοποιείται ήχος με φωνητική ανακοίνωση. Ενεργοποιούνται 2 relay.Το Πρώτο ανάβει άμμεσα τα φώτα του κτηρίου ως προειδοποίηση ,το αναπαράγει τον ανάλογο ήχο ειδοποίησης. 1 Κουμπι διαφορετικού χρώματος για ενεργοποίηση ήχου ανακοίνωσης με μικρόφωνο

💻 Εγκατάσταση Software

  1. Flash Raspberry Pi OS Trixie 32-bit (Raspberry Pi Imager)
  2. sudo apt update && sudo apt install python3-pip mpg123
  3. pip3 install RPi.GPIO gtts
  4. Copy koudounia.py + mp3 files → /home/pi
  5. Test audio: alsamixer (Speaker 100%), mpg123 astiki.mp3

📜 Περιγραφή Κώδικα koudounia.py

🔊 Volume Control

set_volume_93(): 93% Speaker/Mic με amixer πριν κάθε ήχο (σταθερή ένταση).

⚙️ GPIO Setup

  • Buttons: INPUT pull-up (HIGH=ανεκτικό)
  • Relays/LEDs: OUTPUT HIGH=OFF
  • LED running (12): Πράσινο = Ready

⏰ Scheduler

DAY_START=7:30, DAY_END=22:15: Relay1 auto ON/OFF, thread κάθε 30s.

🐕 Watchdog

Thread: /dev/watchdog '1' κάθε 10s → anti-crash.

🔈 Ήχος

  • play_tts_mp3_tts(): TTS ("Αστική Πυρκαγιά") + mp3 + TTS repeat
  • gTTS(lang="el"): On-demand MP3
  • mpg123 quiet playback

🚨 Alarm Sequence

  1. LED κόκκινο ON
  2. Relay1 άμεσο ON (αγνοεί manual/daytime)
  3. Relay2 flash 0.5s
  4. Delay: 4s μέρα / 5s νύχτα (ενισχυτής warmup)
  5. Ήχος → LED OFF

🎤 Mic Button (26)

  • Short: mic.mp3
  • Long 2s+: Toggle manual relay1 (μπλε LED)

🔄 Main Loop

Poll buttons 50ms, debounce 0.3s → ΚΑΘΕ πάτημα δουλεύει!

⚙️ Autostart Service

sudo nano /etc/systemd/system/koudounia.service
[Unit]
Description=ΠΥ Ξάνθης Koudounia Alarm
After=network.target sound.target

[Service]
ExecStart=/usr/bin/python3 /home/pi/koudounia.py
WorkingDirectory=/home/pi
Restart=always
RestartSec=5
User=pi
StandardOutput=journal

[Install]
WantedBy=multi-user.target
sudo systemctl daemon-reload
sudo systemctl enable koudounia
sudo systemctl start koudounia
journalctl -u koudounia -f

🎯 Λειτουργία

  • Κουμπί συμβάν: Flash relay2 → relay1 ON → delay → TTS + mp3 + repeat
  • Mic short: mic.mp3
  • Mic long (2s): Toggle manual relay1 (μπλε LED)
  • Scheduler: Relay1 ON 7:30-22:15
  • LEDs: Κόκκινο=press, Μπλε=manual, Πράσινο=ready
✅ Boot → Auto start!

🔧 Troubleshooting

Γιατί relay1 16 ώρες/ημέρα (07:00-22:00):

Οι απλοί ενισχυτές, ακόμα και οι «επαγγελματικοί» όπως ο PRM120, αναγράφουν στο manual ότι δεν προορίζονται για 24/7 ασταμάτητη λειτουργία.

Αρχική ιδέα: ON/Οff η τροφοδοσια του ενισχυτή με κάθε αναγγελία → Πρόβλημα με warmup του ενισυχτη,οι ήχοι κοβόταν(θελει 3-5s) για να αναψει ο ενισχυτής.

Λύση: ON 16 ώρες → άμεσοι ήχοι. Νύχτα: εδωσα 5s delay για την αναπαραγωγη του ήχου (warmup χρόνος).

Μελλοντική βελτίωση: Phone Ring detector → προ-ενεργοποίηση ενισχυτή μολις "χτυπήσει" το 199 ή το σταθερό.
